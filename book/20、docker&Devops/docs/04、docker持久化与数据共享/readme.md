## docker的持久化存储与数据共享


### <div id="class04-01">01、基础概念</div>
如果我们在container 写数据， 那么只限于这个container, 删除就没有了。                   
有一个这样的需求， 数据库是放在某一个容器里面的， 希望数据不会随着容器的删除而丢失， 希望容器数据库保存的数据可以共享。 

#### Docker持久化数据方案
- 基于本地文件系统的Volume                           
- 基于plugin的Volume, 支持第三方的存储方案， 比如NAS, aws                           

主要是为第一种方案为主。
