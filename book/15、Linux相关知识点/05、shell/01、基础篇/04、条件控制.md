# 提交控制

## if语句

```
if [[ expression ]]
then
   Statement(s) to be executed if expression is true
fi
```

有三种 if ... else 语句：                 
```
if ... fi 语句
if ... else ... fi 语句
if ... elif ... else ... fi 语句
```

示例
```bash
#!/bin/bash/

a=10
b=20
if [ $a == $b ]
then 
	echo "a is equal to b"
elif [ $a -gt $b ]
then
	echo "a is greater to b"
else
	echo "a is less to b"
fi
```
