# 基础vim

## vim起步

命令 | 左右
:- | :-
`i`  | 进入编辑模式
`a`  | 在光标后面开始输入文字
`o`  | 新起一行输入文字
`O`  | 在上一行插入
`v`  | 批量操作
`V`  | 整行的选中
`control + v`  | 块状操作



## 插入模式

### 快速纠错

命令 | 描述
:- |:-
`ctrl + h` | 删除上一个字符
`ctrl + w` | 删除上一个单词
`ctrl + u` | 删除当前行
`ctrl + a` | 行首
`ctrl + b` | 行尾
`gi` | 快速回到最后一次编辑的地方并且插入
