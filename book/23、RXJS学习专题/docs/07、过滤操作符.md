## 过滤操作符的模式


### filter
没有啥好说的

示例1：基础示例                    
```typescript
import { range } from 'rxjs';
import { filter } from 'rxjs/operators';

const source$ = range(1, 5);
const even$ = source$.pipe(
  filter(x => x % 2 === 0),
);

even$.subscribe(value => console.log(value));
/*
结果：
2
4
* */
```
